##Controls:
 * A - Strafe Left
 * D - Strafe Right
 * W - Accelerate
 * Shift - Booster
 * Spacebar - Fire  
 
## Dependencies:
 * pygame - `pip install pygame`
 
## Setup:
 * Clone repository or download files and execute `python main.py` while in  the game directory and you're all set.
 
### Executable:
